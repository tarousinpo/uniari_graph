# uniari_graph - 円グラフ作成アプリ

スプレッドシート形式でデータを入力して、円グラフを簡単に作成できる Web アプリです。

## 機能

- **スプレッドシート入力**: 名前・数値を表形式で入力（行の追加・削除も可能）
- **自動ソート**: グラフ作成時に数値の大きい順に並べ替え
- **カスタム画像**: 上位4名のスライスに好きな画像を使用できます
- **ドラッグ＆ドロップ**: PC内の画像をドラッグ＆ドロップで設定（クリックでファイル選択も可）
- **PNG ダウンロード**: 作成した円グラフを PNG 画像として保存

## 使い方

1. 「データ入力」テーブルに名前と数値を入力する
2. 必要に応じて「＋ 行を追加」でデータを追加する
3. 上位4名に使いたい画像を各スロットにドラッグ＆ドロップ（または クリックで選択）
4. 「📊 円グラフを作成」ボタンを押すとグラフが表示される
5. 「⬇ PNG でダウンロード」でグラフを画像として保存できる

## 開発

```bash
npm install
npm run dev
```

## ビルド
